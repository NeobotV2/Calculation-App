Ziel der App:
Ein Gebäudereiniger soll mobil und am Desktop in wenigen Minuten ein Objekt kalkulieren können. Der Nutzer gibt Räume ein, wählt pro Raum eine Raumart, trägt die Fläche ein, wählt die Reinigungshäufigkeit und die App berechnet automatisch Monatsstunden, Monatspreis und Jahrespreis.

MVP Funktionen:
1. Nutzer kann ein Objekt anlegen
2. Nutzer kann mehrere Räume hinzufügen
3. Jeder Raum hat folgende Felder:
   Bezeichnung optional
   Raumart
   Raumgruppe
   Fläche in m²
   Reinigungshäufigkeit
   Leistungswert in m² pro Stunde
   Kosten pro Monat
4. Leistungswert wird automatisch anhand der Raumart gesetzt
5. Reinigungshäufigkeit soll NICHT als Zyklen pro Jahr eingegeben werden, sondern benutzerfreundlich als:
   5x pro Woche
   3x pro Woche
   2x pro Woche
   1x pro Woche
   14-tägig
   monatlich
6. Die App rechnet intern automatisch mit Monatsfaktoren
7. Nutzer gibt global einen Stundensatz in Euro ein
8. Alle Summen werden live aktualisiert
9. Jeder Raum kann entfernt oder bearbeitet werden
10. Daten bleiben beim Neuladen erhalten

Monatsfaktoren:
5x pro Woche = 21.67
3x pro Woche = 13
2x pro Woche = 8.67
1x pro Woche = 4.33
14-tägig = 2.17
monatlich = 1

Kalkulationslogik:
Für jeden Raum:
Zeit pro Reinigung in Stunden = Fläche / Leistungswert
Monatsstunden = Zeit pro Reinigung * Monatsfaktor
Kosten pro Monat = Monatsstunden * Stundensatz
Jahrespreis = Monatspreis * 12

Gesamtergebnis:
Gesamtfläche = Summe aller Flächen
Monatsstunden gesamt = Summe aller Monatsstunden
Monatspreis gesamt = Summe aller Raumkosten pro Monat
Jahrespreis gesamt = Monatspreis gesamt * 12
Preis pro m² = Monatspreis gesamt / Gesamtfläche
